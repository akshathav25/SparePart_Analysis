# Spare Parts Demand Forecasting

## Overview

This project analyzes historical spare parts sales data and predicts future demand using machine learning. The goal is to help warehouses and aftersales teams plan inventory more efficiently.

## Tools Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

## Workflow

1. Data preprocessing and cleaning
2. Exploratory data analysis
3. Demand forecasting using Linear Regression
4. Identification of fast-moving spare parts

## Visualizations

* Time series chart showing historical demand
* Forecast chart predicting future sales
* Bar chart of top fast-moving spare parts

## Key Insights

* Spare parts demand shows high variability across time.
* Some parts consistently show higher sales than others.
* Forecasting helps estimate future inventory requirements.
